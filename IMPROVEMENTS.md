## 📝 使用說明

### 開啟網站
直接在瀏覽器中打開 `index.html` 即可。

### 優化圖片
```powershell
# 將原始圖片放入 images/originals/
# 執行優化腳本
cd k-pulse
node scripts/optimize-images.js
```

### 主題切換
點擊右上角的 ☀️/🌙 按鈕切換日夜模式（自動保存偏好）。

### 搜尋功能
在偶像總覽頁面使用搜尋框，支援：
- 團體名稱搜尋
- 標籤搜尋
- 即時過濾
- Esc 快速清除
