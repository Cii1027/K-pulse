NEXZ

出道：2024 | 粉絲名：NEXT

團體簡介

NEXZ 是JYP與Sony Music Japan共同打造的日本男團，透過選秀節目《Nizi Project 2》誕生。以強大的演唱和舞蹈實力展現新世代男團魅力，代表作包含《Ride the Vibe》和《Miracle》。

代表作品

《Ride the Vibe》- NEXZ 出道代表作，展現強大實力

成員與個人經歷

Tomoya (トモヤ)

職務：隊長 / 主唱 / 主舞

  • 擔任隊長，以穩定領導力和全能實力著稱。
  • 在《Nizi Project 2》中展現優秀表現。

Haru (ハル)

職務：主唱 / 副舞

  • 以清亮嗓音和穩定演唱實力著稱。
  • 參與團體主要演唱部分，展現歌唱技巧。

Yuki (ユウキ)

職務：主舞 / 副唱

  • 以精湛舞蹈實力和舞台表現力著稱。
  • 參與多個舞蹈編排，展現個人風格。

Seita (セイタ)

職務：主Rapper / 副舞

  • 以強烈Rap風格和舞台魅力著稱。
  • 負責團體Rap部分，展現音樂才華。

Hyui (ヒュイ)

職務：副唱 / 副舞

  • 韓國籍成員，以穩定表現力和親和力著稱。
  • 參與團體活動，展現多樣魅力。

Yuhi (ユウヒ)

職務：副唱 / 副Rapper

  • 以多樣音樂風格和舞台魅力著稱。
  • 參與團體演出，展現個人特色。

So Geon (ソ・ゴン)

職務：副唱 / 副舞 / 忙內

  • 團隊最年輕成員，韓國籍，以潛力和活力著稱。
  • 參與團體演出，展現年輕魅力。
