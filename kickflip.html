KickFlip

出道：2025 | 粉絲名：待公布

團體簡介

KickFlip 是JYP旗下的新世代男團，以年輕活力和全能實力受到期待。成員均具備演唱、舞蹈、Rap等多樣才能，展現第五代偶像團體的魅力。

成員與個人經歷

Keiju (ケイジュ)

職務：隊長 / 主唱 / 主舞

  • 擔任隊長，以穩定領導力和全能實力著稱。
  • 參與團體訓練，展現優秀表現。

Amaru (アマル)

職務：主唱 / 副舞

  • 以清亮嗓音和穩定演唱實力著稱。
  • 參與團體主要演唱部分，展現歌唱技巧。

Leejun (イジュン)

職務：主舞 / 副唱

  • 以精湛舞蹈實力和舞台表現力著稱。
  • 參與多個舞蹈編排，展現個人風格。

Seren (セレン)

職務：主Rapper / 副舞

  • 以強烈Rap風格和舞台魅力著稱。
  • 負責團體Rap部分，展現音樂才華。

Masaya (マサヤ)

職務：副唱 / 副舞

  • 以穩定表現力和親和力著稱。
  • 參與團體活動，展現多樣魅力。

Donghwa (ドンファ)

職務：副唱 / 副Rapper

  • 韓國籍成員，以多樣音樂風格著稱。
  • 參與團體演出，展現個人特色。

Ren (レン)

職務：副唱 / 副舞 / 忙內

  • 團隊最年輕成員，以潛力和活力著稱。
  • 參與團體演出，展現年輕魅力。
